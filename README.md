## 📊 Student Grade Tracker (Java GUI)

This project is a simple Java-based application developed to manage and analyze student grades. It provides a user-friendly graphical interface built using Swing, allowing users to easily input, view, and manage student records.

The application allows the user to enter student details such as name, roll number, and percentage. Based on the entered percentage, the system automatically calculates the corresponding grade (A, B, C, or Fail). Both percentage and grade are displayed in a structured table format.

Users can also delete student records using a combination of name and roll number, ensuring accuracy since names may not be unique. The system prevents duplicate roll numbers to maintain data integrity.

Additionally, the project calculates key statistics such as average, highest, and lowest scores, giving a quick overview of student performance.

### 🔑 Features
- Add student (Name, Roll No, Percentage)
- Automatic grade calculation (A/B/C/Fail)
- Display data using JTable (structured format)
- Delete student using Name + Roll No
- Prevent duplicate roll numbers
- Summary statistics (average, highest, lowest)

### 🛠 Technologies Used
- Java
- Swing (GUI)
- ArrayList (data storage)

This project demonstrates the use of Object-Oriented Programming, event handling, and GUI design in Java.
